# UExP-FNN-U full carbonate system dataset

This section of the repository holds the config files used with the OceanICU uncertainty framework to generate the UExP-FNN-U full carbonate system dataset. The UExP-FNN-U receives regular updates and all versions are linked below. The repository for each version contains a full changelog and description of the methodology to generate the data.
